Inventory Location Restrictions and Access Control Documentation

1- Set User Right for User

- If the user has the right for see all locations, then the user sees all locations and its operations and it is also possible that some locations may not be visible to the user.

2- Define User in Location

- Any user can only see the location specified for him, the quantities of the product inside it, and the operations of this store, and the admin has authority over each store and its operations

3- Location in User

- When we define users in location then location store here automatically.

4- Stock with Restriction

- When the user has not right for access all locations then it sees its own location product stock

5- Stock without Restriction for admin

6- The admin can create any operation type for any store

7- The users cannot perform any operation type on a store for which he has not specified

8-The user can make a receipt only at the stores designated for him

9- Select Allowed Location in Source Location as Well as Destination Location

- User can select only own location in source location as well as destination location

10- The admin can make a receipt at any store

11- The admin can make a transfer from Source Location to Destination Location in any store
